    Quick start
  </h2>
  <p className="mt-3 text-base text-gray-600 dark:text-gray-400 text-center max-w-2xl mx-auto">
    Get OpenGraphs running in minutes with our one-line installer
  </p>

  <div className="mt-10">
    <Steps>
      <Step title="Install OpenGraphs">
        Run the installer script to download the latest release:
        
        ```bash
        curl -fsSL https://raw.githubusercontent.com/vyomakesh0728/opengraphs/main/scripts/install.sh | bash
        ```
        
        Or run directly with npx (no install required):
        
        ```bash
        npx -y opengraphs-cli@latest
        ```
      </Step>

      <Step title="Track your first experiment">
        Run your training script with OpenGraphs to start tracking metrics:
        
        ```bash
        og run train.py --runtime local
        ```
        
        The TUI will launch automatically, displaying real-time metrics from TensorBoard events.
      </Step>

      <Step title="Explore the interface">
        Navigate through the TUI tabs:
        - **Graphs**: View metrics visualizations
        - **Logs**: Monitor training output
        - **Procs**: Track system resources
        - **Chat**: Interact with the AI agent
        
        <Tip>
          Press `q` to quit the TUI, or use keyboard shortcuts to navigate between tabs.
        </Tip>
      </Step>

      <Step title="Enable AI diagnostics">
        Launch runs with autonomous AI agent support for automatic issue detection:
        
        ```bash
        og run train.py --auto autonomous
        ```
        
        The agent monitors your training metrics and suggests fixes when it detects plateaus or errors.
      </Step>
    </Steps>
  </div>
</div>
